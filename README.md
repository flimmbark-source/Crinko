# Crinko - Samurai Duel Prototype

A mobile-first, serious samurai-inspired simultaneous card dueler prototype.

## Run

```bash
npm install
npm run dev
```

Then open the local Vite URL.

## Test

```bash
npm test
```

## Prototype loop

1. Start a duel vs AI.
2. Each exchange: draw 5 and keep 3.
3. Resolve 3 simultaneous beats.
4. Continue until KO or double KO draw.
5. Pick a lightweight post-duel reward and restart.
