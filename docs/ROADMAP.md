# Roadmap

## Current prototype (implemented)
- Deterministic event-log combat engine.
- Stone and Wind starter schools with passives and signatures.
- Shared pool cards for attacks, defense, tricks, flow disruption, and delayed effects.
- Mobile-first single-screen duel UI.
- AI that evaluates range, flow, pressure, and occasional dramatic reads.
- Lightweight post-duel reward choice.
- Core rules tests for speed, initiative, range, guard, armed effects, flow, exchange cleanup, and KO handling.

## Next improvements
1. Better deck shaping in reward layer (real add/remove/upgrade persistence).
2. Draft-style pre-duel card picks.
3. Richer AI personality tuning by school.
4. Animation pass on reveal lane while retaining authoritative engine state.
5. More signature cards and archetype-specific win lines.
